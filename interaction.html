  Let's Learn some basic Music Theory  
 

Note Value

Four types of the note for beginner

 
     
  1.  Eighth note: A note that is half beat. In music programming half beat could equal 250ms.  
 

 
     
  2. Quarter note: A note that have one beat. In music programming one beat could equal 500ms  
 

 
     
  3. Half Note: A note that have two beats. In music programming two beats could equal 1000ms.  
 

 
     
  4. Whole Note: A note that have four notes. In music programming four beats could equal 2000ms.  
 

 
     
     
 

Note Name

In music, we have something called music alphabet which start with C. The order for music alphabet is C, D, E, F, G, A, B. That is all the simple note name we need to learn as beginner.

As you can see from the keyboard above, other than C, D, E, F, G is what we call a white key, we would have something called black key. If the black key is on the right side of one white key the note name would be the name of the white key and adding a sharp.

  
 

For example, if we have a C and the black key that on the right side of the C will be name C# (sharp).

  
 
C
C#/ Bb
  
     
 

If the black key is on the left side of one white key, the note name would be the name of the white key and adding a flat. For example if we have D, and the black key that on the left side of the D will be name Db (flat).

  
   
D
D#/ Eb
  
     
   

Now you can see, one black would have two note names, it can be flat and sharp depend on the music.

 

There are two special circumstances for the sharp and flat note. Before we get in to that we will talk about one new concept. The half step and whole step. Half step is the closest distance between two key, for example C to C# is a half-step where two keys are closely together with no space in between. The whole step is the distance of two half step together, for example C to D is a whole step where it contains two groups of half step C to C# and C# to D.

Now we can go back to the note naming, the sharp would be any note that move half step to the right. The flat would be any note that move half step to the left. Keep this in mind, if we have C, and we want to have a C flat. This is the special circumstance where flat note is no longer a black key. If C moves half step to the left, we would get a B and that can also be C flat. Same with the sharp the special circumstance would be E sharp. If we move half step to the right for E, we will get F and that can also be E sharp.
